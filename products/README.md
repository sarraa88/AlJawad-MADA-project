# Products

This folder contains the final outputs of the lung cancer risk analysis project.

## Subfolders

- `report/`
  - Contains the Quarto report (`report.qmd`) and the rendered HTML output.
  - This provides a web-based presentation of the analysis, structured in a standard scientific format.

- `manuscript/`
  - Contains the Quarto manuscript (`Manuscript.qmd`) and associated output files (e.g., Word).
  - This presents the analysis in a manuscript-style format suitable for written reporting.

Both formats integrate results from the full analysis pipeline, including descriptive analysis, statistical modeling, and predictive modeling.

All outputs in this folder are generated programmatically and can be reproduced by running the analysis scripts and rendering the corresponding Quarto documents.
