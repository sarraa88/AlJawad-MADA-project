# data


This folder contains all data used in the lung cancer risk analysis project.

## Folder structure

- `raw-data/`
  - Contains the original dataset: `lung_cancer.csv`
  - This file is not modified.

- `processed-data/`
  - Contains the cleaned dataset: `lung_clean.rds` (or `.csv`)
  - Generated by `code/processing-code/01_load_clean.R`

All processed files can be reproduced by running the scripts in the `code/processing-code/` directory.
